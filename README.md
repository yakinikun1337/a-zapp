# Bangkok A-Z Drinking Game

Ready-to-deploy React/Vite app.

## Deploy with GitHub + Vercel

1. Unzip this folder.
2. Create a new GitHub repository.
3. Upload all files from this folder to the repository.
4. Open Vercel.
5. Click Add New Project.
6. Import the GitHub repository.
7. Click Deploy.

## Add to Samsung Galaxy S21

1. Open the deployed Vercel link in Chrome.
2. Tap the three-dot menu.
3. Tap Add to Home screen.
4. Open it from your phone like an app.

## Safety

Share drinks, hydrate, eat food, skip rounds freely, and do not drive.
